# Bangkok Urban Planner 3D

Interactive 3D visualization of urban planning solutions for Bangkok, Thailand.

## Features

- **3D City Model** — Procedurally generated Bangkok-inspired urban layout with the Chao Phraya River
- **Flood Resilience** — Visualize flood-prone zones and sponge city solutions
- **Transit Network** — Explore current and proposed rail expansion (25km → 433km by 2032)
- **Green Spaces** — See the gap between current (5.6 sqm/capita) and target (27 sqm/capita) green space
- **15-Minute City** — Integrated multimodal urban planning visualization
- **Air Quality** — PM2.5 overlay with green corridor mitigation

## Controls

- **Click & drag** to orbit the 3D view
- **Scroll** to zoom in/out
- **Navigation tabs** to switch between scenarios
- **Challenge cards** on the sidebar for detailed Bangkok statistics
- **Sliders** to adjust green cover, transit coverage, and flood risk levels

## Tech Stack

- Three.js for 3D rendering
- Procedural city generation with simplex noise
- Vanilla JavaScript, no build tools needed

## Usage

Simply open `index.html` in a browser. No build step required.
